# V1.0.3 UI层级冲突修复报告

## 问题描述
用户反馈：位置权限弹窗与匹配页面底部的"开始匹配"按钮重叠冲突

## 根本原因
使用 `showModalBottomSheet` 从底部弹出的对话框，会与页面底部的按钮产生层级冲突和视觉重叠

## 解决方案
将所有 `showModalBottomSheet` 改为 `showDialog`，使用居中弹窗替代底部弹窗，避免层级冲突

## 修复文件清单

### 1. permission_manager.dart
**修复内容：**
- ✅ `_showPermissionDialog()` - 权限请求对话框
- ✅ `_showGoToSettingsDialog()` - 跳转设置对话框  
- ✅ `_showLocationSelectionDialog()` - 位置选择对话框

**改动：**
```dart
// 修改前
showModalBottomSheet(
  context: context,
  backgroundColor: Colors.transparent,
  builder: (context) => Container(...)
)

// 修改后
showDialog(
  context: context,
  builder: (context) => Dialog(
    backgroundColor: AppColors.cardBg,
    shape: RoundedRectangleBorder(
      borderRadius: BorderRadius.circular(24),
    ),
    child: Padding(...)
  )
)
```

## UI优化细节

### 对话框统一规范
1. **背景色**：`AppColors.cardBg`
2. **圆角**：`BorderRadius.circular(20-24)`
3. **内边距**：`EdgeInsets.all(24-32)`
4. **按钮样式**：统一使用 `TextButton` + 圆角12
5. **文字样式**：
   - 标题：`fontSize: 18-20, fontWeight: w400`
   - 内容：`fontSize: 15, fontWeight: w300`
   - 按钮：`fontSize: 15, fontWeight: w300`

### 视觉层级
- **Dialog**：居中显示，带半透明遮罩
- **优势**：不会与底部按钮冲突，视觉更清晰
- **用户体验**：更符合主流应用习惯
