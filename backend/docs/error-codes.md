# Error Code Catalog (v1)

## Format
- HTTP status + business code
- Response:
```json
{
  "code": "AUTH_OTP_INVALID",
  "message": "验证码错误",
  "detail": "请重新输入",
  "request_id": "req_xxx"
}
```

## Auth
- `AUTH_OTP_INVALID` (400)
- `AUTH_OTP_EXPIRED` (400)
- `AUTH_TOKEN_EXPIRED` (401)
- `AUTH_TOKEN_INVALID` (401)
- `AUTH_DEVICE_FORBIDDEN` (403)

## User/Profile
- `USER_NOT_FOUND` (404)
- `UID_NOT_FOUND` (404)
- `UID_FORMAT_INVALID` (400)
- `PROFILE_UPDATE_INVALID` (400)

## Social
- `FRIEND_REQUEST_DUPLICATE` (409)
- `FRIEND_REQUEST_NOT_FOUND` (404)
- `FRIEND_ALREADY_EXISTS` (409)
- `BLOCKED_RELATION` (403)
- `USER_BLOCKED` (403)

## Match
- `MATCH_QUOTA_EXCEEDED` (429)
- `MATCH_NOT_FOUND` (404)
- `MATCH_CANCELLED` (409)

## Chat
- `THREAD_NOT_FOUND` (404)
- `MESSAGE_SEND_FAILED` (500)
- `MESSAGE_NOT_FOUND` (404)
- `MESSAGE_RECALL_EXPIRED` (409)
- `MESSAGE_BURNED` (410)
- `THREAD_EXPIRED` (409)

## Policy
- `POLICY_UNLOCK_REQUIRED` (403)
- `POLICY_SCORE_INSUFFICIENT` (403)
- `POLICY_DURATION_INSUFFICIENT` (403)
- `VOICE_NOT_UNLOCKED` (403)

## Storage/Infra
- `UPLOAD_TOKEN_FAILED` (500)
- `RATE_LIMITED` (429)
- `SERVICE_UNAVAILABLE` (503)
- `INTERNAL_ERROR` (500)
