### 2. 完整流程

#### 选择头像流程

```dart
// 1. 显示来源选择对话框
final source = await showImageSourceDialog(context);
// 用户选择：拍照 / 相册 / 取消

// 2. 请求对应权限
if (source == ImageSource.camera) {
  hasPermission = await PermissionManager.requestCameraPermission(context);
} else {
  hasPermission = await PermissionManager.requestPhotosPermission(context);
}

// 3. 选择图片
final XFile? image = await ImagePicker().pickImage(
  source: source,
  maxWidth: 1024,   // 头像限制1024x1024
  maxHeight: 1024,
  imageQuality: 90,
);

// 4. 保存到应用目录
final appDir = await getApplicationDocumentsDirectory();
final targetPath = '${appDir.path}/avatar_${timestamp}.jpg';
final savedFile = await sourceFile.copy(targetPath);

// 5. 持久化存储路径
final prefs = await SharedPreferences.getInstance();
await prefs.setString('user_avatar_path', savedFile.path);

// 6. 删除旧文件
if (oldPath != null) {
  await File(oldPath).delete();
}
```

### 3. 权限管理

#### 相机权限

```dart
static Future<bool> requestCameraPermission(BuildContext context) async {
  final status = await Permission.camera.status;
  
  if (status.isGranted) return true;
  
  if (status.isDenied) {
    // 显示权限说明对话框
    final shouldRequest = await _showPermissionDialog(
      context,
      title: '开启相机权限',
      content: '开启相机权限后，你可以拍摄照片设置头像',
      icon: Icons.camera_alt_outlined,
    );
    
    if (shouldRequest == true) {
      final result = await Permission.camera.request();
      return result.isGranted;
    }
  }
  
  if (status.isPermanentlyDenied) {
    // 引导去系统设置
    await _showGoToSettingsDialog(context);
  }
  
  return false;
}
```

## 🎯 用户体验

### 权限请求流程

#### 首次请求

```
用户点击"更换头像" 
→ 显示来源选择（拍照/相册）
→ 用户选择"拍照"
→ 显示权限说明对话框
   "开启相机权限后，你可以拍摄照片设置头像"
   [暂不开启] [去开启]
→ 用户点击"去开启"
→ 系统权限弹窗
→ 用户授权
→ 打开相机
```

#### 权限被拒绝

```
用户拒绝权限
→ 提示"需要相应权限才能选择图片"
→ 返回
```

#### 权限永久拒绝

```
用户永久拒绝权限
→ 显示引导对话框
   "相机权限已关闭"
   "请在系统设置中开启相机权限"
   [取消] [去设置]
→ 用户点击"去设置"
→ 跳转到系统设置页面
```

## 🚀 使用说明

### 开发者

```dart
// 1. 选择头像
final avatarFile = await ImageUploadService.pickAvatar(context);
if (avatarFile != null) {
  // 头像已保存，自动持久化
  setState(() {
    _avatarPath = avatarFile.path;
  });
}

// 2. 获取头像路径
final avatarPath = await ImageUploadService.getAvatarPath();

// 3. 显示头像
if (avatarPath != null && File(avatarPath).existsSync()) {
  Image.file(File(avatarPath));
}

// 4. 删除头像
await ImageUploadService.clearAvatar();
```

### 用户

1. **设置头像**
   - 进入"我的"页面
   - 点击头像
   - 选择"拍照"或"从相册选择"
   - 授权权限
   - 选择图片
   - 完成

2. **管理头像**
   - 进入"设置"页面
   - 点击"头像管理"
   - 选择"更换头像"或"删除头像"
