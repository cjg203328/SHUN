# 瞬 FlashChat V1.0.0

**24小时限时匿名社交应用**

---

## 📱 项目简介

瞬聊是一款极简、克制的匿名社交应用。在24小时的限定时间内，与陌生人建立真实而短暂的连接。

### 核心理念
- **限时性**：24小时后对话自动消失
- **匿名性**：渐进式身份解锁
- **克制性**：每日20次匹配机会
- **纯粹性**：极简设计，专注对话

## 🚀 快速开始

### 开发环境


