### 2.2 技术选型

#### 2.2.1 状态管理
**选择**：Provider  
**原因**：
- 官方推荐，稳定可靠
- 学习曲线平缓
- 性能优秀
- 社区支持好

**实现方式**：
```dart
// 在main.dart中注册Provider
MultiProvider(
  providers: [
    ChangeNotifierProvider(create: (_) => AuthProvider()),
    ChangeNotifierProvider(create: (_) => MatchProvider()),
    ChangeNotifierProvider(create: (_) => ChatProvider()),
  ],
  child: MyApp(),
)
```

#### 2.2.2 路由管理
**选择**：GoRouter  
**原因**：
- 声明式路由
- 支持深度链接
- 类型安全
- 易于维护

**路由配置**：
```dart
final router = GoRouter(
  initialLocation: '/splash',
  routes: [
    GoRoute(path: '/splash', builder: (context, state) => SplashScreen()),
    GoRoute(path: '/login', builder: (context, state) => LoginScreen()),
    GoRoute(path: '/main', builder: (context, state) => MainScreen()),
    GoRoute(path: '/chat/:id', builder: (context, state) => ChatScreen(
      threadId: state.pathParameters['id']!,
    )),
  ],
);
```

#### 2.2.3 本地存储
**选择**：SharedPreferences  
**原因**：
- 轻量级键值存储
- 适合存储简单配置
- 异步操作
- 跨平台支持

**使用场景**：
- 登录状态
- 用户信息
- 匹配次数
- 最后重置时间

**核心代码**：

```dart
class AuthProvider extends ChangeNotifier {
  bool _isLoggedIn = false;
  String? _phone;
  
  Future<void> login(String phone, String code) async {
    // Mock验证
    if (code == '123456') {
      _isLoggedIn = true;
      _phone = phone;
      await StorageService.savePhone(phone);
      notifyListeners();
    }
  }
  
  Future<void> logout() async {
    _isLoggedIn = false;
    _phone = null;
    await StorageService.clearAll();
    notifyListeners();
  }
}
```

### 3.2 匹配模块

#### 3.2.1 MatchProvider
**职责**：
- 管理匹配次数
- 处理匹配逻辑
- 生成随机用户

**核心功能**：
```dart
class MatchProvider extends ChangeNotifier {
  int _matchCount = 20;
  User? _matchedUser;
  bool _isMatching = false;
  
  // 开始匹配
  Future<void> startMatch() async {
    _isMatching = true;
    notifyListeners();
    
    await Future.delayed(Duration(milliseconds: 2500));
    
    _matchedUser = _generateRandomUser();
    _isMatching = false;
    notifyListeners();
  }
  
  // 消耗次数
  void consumeMatch() {
    if (_matchCount > 0) {
      _matchCount--;
      notifyListeners();
    }
  }
  
  // 每日重置
  void checkAndResetDaily() {
    final now = DateTime.now();
    final lastReset = StorageService.getLastResetTime();
    
    if (lastReset == null || 
        now.day != lastReset.day ||
        now.hour >= 9 && lastReset.hour < 9) {
      _matchCount = 20;
      StorageService.saveLastResetTime(now);
      notifyListeners();
    }
  }
}
```

### 3.3 聊天模块

#### 3.3.1 ChatProvider
**职责**：
- 管理对话列表
- 处理消息收发
- 管理解锁状态

**核心功能**：
```dart
class ChatProvider extends ChangeNotifier {
  Map<String, ChatThread> _threads = {};
  Map<String, List<Message>> _messages = {};
  
  // 发送消息
  void sendMessage(String threadId, String content) {
    final message = Message(
      id: uuid.v4(),
      threadId: threadId,
      content: content,
      isMe: true,
      timestamp: DateTime.now(),
    );
    
    _messages[threadId]?.add(message);
    notifyListeners();
    
    // Mock自动回复
    _mockAutoReply(threadId);
  }
  
  // Mock自动回复
  Future<void> _mockAutoReply(String threadId) async {
    await Future.delayed(Duration(seconds: 2));
    
    final replies = ['哈哈哈', '是吗', '然后呢', '有意思', '我也这么觉得'];
    final reply = replies[Random().nextInt(replies.length)];
    
    final message = Message(
      id: uuid.v4(),
      threadId: threadId,
      content: reply,
      isMe: false,
      timestamp: DateTime.now(),
    );
    
    _messages[threadId]?.add(message);
    _threads[threadId]?.unreadCount++;
    notifyListeners();
  }
  
  // 更新解锁等级
  void updateUnlockLevel(String threadId, int level) {
    _threads[threadId]?.unlockLevel = level;
    notifyListeners();
  }
}
```

## 五、数据存储

### 5.1 本地存储

#### 5.1.1 StorageService
**文件**：`lib/services/storage_service.dart`  
**功能**：
- 封装SharedPreferences
- 提供统一的存储接口

**核心方法**：
```dart
class StorageService {
  static SharedPreferences? _prefs;
  
  static Future<void> init() async {
    _prefs = await SharedPreferences.getInstance();
  }
  
  static Future<void> savePhone(String phone) async {
    await _prefs?.setString('phone', phone);
  }
  
  static String? getPhone() {
    return _prefs?.getString('phone');
  }
  
  static Future<void> clearAll() async {
    await _prefs?.clear();
  }
}
```

### 5.2 数据模型

```dart
class User {
  final String id;
  final String nickname;
  final String avatar;

  final String distance;
  
  User({
    required this.id,
    required this.nickname,
    required this.avatar,

  Map<String, dynamic> toJson() => {
    'id': id,
    'nickname': nickname,
    'avatar': avatar,
    'status': status,
    'distance': distance,
```
