# 瞬 V1.0.1

**24小时限时匿名社交应用**

---

## 📱 项目简介

瞬是一款极简、克制的匿名社交应用。在24小时的限定时间内，与陌生人建立真实而短暂的连接。

### 核心理念
- **限时性**：24小时后对话自动消失
- **匿名性**：渐进式身份解锁
- **克制性**：每日20次匹配机会
- **纯粹性**：极简设计，专注对话

## ✨ 核心功能

### 1. 随机匹配
- 每日20次匹配机会
- 光球呼吸动画
- 招呼语系统（二选一）
- 取消匹配功能

### 2. 限时聊天
- 24小时倒计时
- 渐进式解锁（15分钟/30分钟）
- 消息收发
- 长按复制/撤回

### 3. 好友系统
- 手动请求+同意机制
- 30分钟后可添加好友
- 好友聊天无时间限制
- 设置备注功能

### 4. 消息管理
- 对话列表
- 未读提示
- 滑动删除（二次确认）

### 5. 个人中心
- 头像设置
- 昵称修改
- 状态选择
- 设置页面

### 6. 权限管理
- 通知权限
- 相机权限
- 相册权限
- 麦克风权限
