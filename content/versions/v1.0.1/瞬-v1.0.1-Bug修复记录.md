### BUG-002: 登录页错误提示不统一 ✅
**优先级**: P1  
**发现时间**: 2026-02-23 10:00  
**修复时间**: 2026-02-23 10:30  

**问题描述**:  
登录页使用SnackBar而非AppToast，不符合统一设计

**复现步骤**:  
1. 打开登录页
2. 输入错误验证码
3. 点击登录
4. 查看错误提示样式

**问题原因**:  
未使用统一的AppToast组件

**修复方案**:  
```dart
// 修改前
ScaffoldMessenger.of(context).showSnackBar(
  const SnackBar(content: Text('请输入正确的手机号')),
);

// 修改后
AppToast.show(context, '请输入正确的手机号');
AppToast.show(context, '验证码错误，请输入123456', isError: true);
```

**验证结果**: ✅ 已修复
