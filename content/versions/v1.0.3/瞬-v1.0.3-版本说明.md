## 📦 技术细节

### 新增依赖
```yaml
image_picker: ^1.0.7  # 图片选择（为后续功能准备）
image: ^4.1.7         # 图片处理（为后续功能准备）
path_provider: ^2.1.2 # 路径管理（为后续功能准备）
```

### 数据模型更新
- Message模型新增字段：
  - `type`: 消息类型（text/image）
  - `imagePath`: 图片路径
  - `isBurnAfterReading`: 是否阅后即焚
  - `isRead`: 是否已读
  - `imageQuality`: 图片质量

- AuthProvider新增字段：
  - `nickname`: 用户昵称
  - `avatar`: 用户头像
  - `status`: 用户状态

### 存储服务扩展
- 新增用户资料存储方法
- 新增位置权限缓存管理
- 优化数据持久化逻辑
