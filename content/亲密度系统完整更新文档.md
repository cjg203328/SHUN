## 📊 亲密度系统规则

### 解锁时间线
```
匹配成功（0分）
  ↓ 约5分钟聊天
20分 → 解锁头像 ✨
  ↓ 约5分钟聊天
50分 → 解锁昵称 ✨
  ↓ 约10分钟聊天
100分 → 解锁个人签名 ✨
  ↓ 约10分钟聊天
150分 → 解锁主页 ✨
  ↓ 约5分钟聊天
200分 → 解锁主页背景 ✨
  ↓ 约5分钟聊天
250分 → 可以互关 ✨✨
```

### 亲密度计算规则
| 行为 | 获得分数 | 说明 |
|------|---------|------|
| 发送短消息（<5字） | +1分 | 基础分数 |
| 发送中等消息（5-20字） | +2分 | 基础分数 |
| 发送长消息（20-50字） | +3分 | 基础分数 |
| 发送超长消息（>50字） | +5分 | 基础分数 |
| 接收消息 | +1~5分 | 同发送规则 |
| 首次对话 | +10分 | 一次性奖励 |
| 连续对话（5分钟内） | +3分 | 每次互动 |
| 深夜聊天（23:00-6:00） | +2分 | 每条消息 |

### 取关逻辑
```
好友A取关好友B
  ↓
B的聊天会话标记为 isUnfollowed = true
  ↓
B发送第1条消息 ✅ 可以发送
  ↓
B发送第2条消息 ✅ 可以发送
  ↓
B发送第3条消息 ✅ 可以发送
  ↓
B尝试发送第4条 ❌ 被拦截
  ↓
显示提示："等待对方确认继续聊天"
  ↓
A点击"确认继续聊天"
  ↓
isUnfollowed = false，恢复正常 ✅
```

## 🎨 UI设计特点

### 1. 极简柔和
- 亲密度进度条：橙色渐变，无边框
- 解锁提示：小字号，灰色，不突兀
- 取关提示：半透明红色背景，柔和过渡

### 2. 视觉反馈
- 头像：未解锁半透明，已解锁正常
- 昵称：未解锁灰色"神秘人"，已解锁白色真名
- 亲密度变化：飘起动画，橙色爱心

### 3. 信息层次
- 主要信息：昵称、消息内容
- 次要信息：亲密度等级、解锁提示
- 辅助信息：亲密度分数、进度条

### 4. 交互自然
- 进度条自动更新，无需手动刷新
- 解锁瞬间无弹窗，只显示小提示
- 取关操作需要二次确认
