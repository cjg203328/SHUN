## 一、产品概述

### 1.1 产品定位
瞬是一款极简、克制的24小时限时匿名社交应用。

### 1.2 核心理念
- **限时性**: 24小时后对话自动消失
- **匿名性**: 渐进式身份解锁
- **克制性**: 每日20次匹配机会
- **纯粹性**: 极简设计，专注对话

### 1.3 目标用户
- 年龄: 18-35岁
- 特征: 喜欢深度交流，追求真实连接
- 场景: 深夜失眠、情绪低落、想找人聊天

---

## 二、功能架构

```
瞬 APP
├── 启动模块（启动页、登录注册）
├── 匹配模块（随机匹配、招呼语）
├── 聊天模块（消息收发、渐进式解锁）
├── 好友模块（好友请求、好友列表）
├── 消息模块（对话列表、未读提示）
└── 个人模块（个人主页、设置、权限）
```

## 三、核心功能详细设计

### 3.1 启动与登录

#### 启动页
- 纯黑背景 + 光球呼吸动画
- 品牌名"瞬"（56号超轻字重，16px字间距）
- 2.5秒后自动跳转

#### 登录注册
- 手机号输入（11位数字）
- 验证码输入（Mock: 123456）
- 60秒倒计时
- 错误提示使用AppToast（底部浮动）

---

### 3.2 匹配系统

#### 匹配页面布局
```
┌─────────────────────┐
│      品牌名"瞬"      │
│    情感文案区域      │
│   ◉ 光球动画 ◉      │
│   今日剩余 20 次    │
│   [开始匹配]按钮    │
└─────────────────────┘
```

#### 匹配逻辑
```
点击"开始匹配" → 匹配中（2.5秒）
  ↓
  可点"取消" → 不消耗次数 ✅
  ↓
匹配成功（弹出卡片） → 立即消耗1次 ✅
  ↓
  点"算了" → 关闭卡片
  点"打招呼" → 进入招呼页
```

#### 招呼语系统
- 快捷语和自定义输入二选一
- 选择快捷语时隐藏输入框
- 开始输入时隐藏快捷语

**快捷语选项**:
- "嗨，很高兴认识你"
- "你好呀，聊聊天吧"
- "Hi，在做什么呢"

---

### 3.3 聊天功能

#### 页面结构
```
┌─────────────────────────────┐
│ ← [头像] 昵称 [更多]        │
│   15分钟解锁头像             │
├─────────────────────────────┤
│  [对方消息]                  │
│           [我的消息]         │
├─────────────────────────────┤
│ [+] [输入框...] [发送]      │
└─────────────────────────────┘
```

#### 渐进式解锁
```
0分钟: 匹配成功
  ↓
15分钟: 解锁头像
  ↓
30分钟: 解锁主页 + 可添加好友
  ↓
24小时: 对话消失（除非成为好友）
```

#### 消息操作
- 长按消息 → 弹出菜单
  - 复制（所有消息）
  - 撤回（2分钟内，仅自己的消息）

#### 更多菜单
- 查看个人主页（30分钟后）
- 语音通话（好友或30分钟后）
- 添加好友（30分钟后）
- 举报用户
- 删除对话

---

### 3.4 好友系统

#### 好友流程
```
陌生人匹配 → 聊天15分钟 → 解锁头像
  ↓
聊天30分钟 → 可添加好友
  ↓
发送好友请求（可选附言）
  ↓
对方接受 → 成为好友
  ↓
好友聊天（无时间限制）
```

#### 好友请求
- 触发条件: 聊天30分钟后
- 附言: 可选，最多50字
- 通知: 好友Tab右上角红点

#### 好友列表
- 显示: 头像、昵称（或备注）、成为好友时间
- 操作: 点击聊天、长按设置备注/删除

#### 好友聊天特点
- 无24小时限制
- 完全解锁状态
- 可随时聊天

---

### 3.5 消息列表

#### 列表内容
- 用户头像
- 用户昵称
- 最后一条消息
- 消息时间
- 未读数量（红点）
- 倒计时（陌生人对话）

#### 操作
- 点击 → 进入聊天
- 左滑 → 删除（二次确认）

**隐私设置**
- 黑名单管理
- 隐身模式
- 位置权限

**通知设置**
- 消息通知开关
- 好友请求通知
- 震动/声音开关

**关于**
- 关于我们
- 隐私政策
- 用户协议
- 版本信息（V1.0.1）

---

### 3.7 权限管理

#### 权限类型

| 权限 | 用途 | 请求时机 | 保留策略 |
|------|------|----------|----------|
| 通知 | 接收消息 | 首次登录 | 永久保留 |
| 相机 | 拍照头像 | 点击拍照 | 永久保留 |
| 相册 | 选择照片 | 点击相册 | 永久保留 |
| 麦克风 | 语音通话 | 发起通话 | 永久保留 |
| 位置 | 匹配附近 | 每次匹配 | 每次确认 |

#### 权限请求流程
```
触发功能 → 检测权限
  ↓
未授权 → 显示说明弹窗
  ↓
点击"去开启" → 系统权限弹窗
  ↓
授权成功 → 执行功能
  ↓
永久拒绝 → 引导去设置
```

#### 权限弹窗设计
- 大图标（权限类型）
- 标题（如"开启相机权限"）
- 说明文案
- 按钮: "暂不开启" / "去开启"
- 底部弹出，半透明背景，圆角24px

## 四、用户流程

### 4.1 新用户首次使用
```
启动 → 登录 → 匹配 → 聊天 → 15分钟解锁头像 
→ 30分钟添加好友 → 成为好友
```

### 4.2 老用户日常使用
```
启动 → 查看消息 → 继续聊天
  或
启动 → 开始匹配 → 认识新人
  或
启动 → 查看好友 → 与好友聊天
```

## 五、设计规范

### 5.1 颜色系统
- **纯黑**: #000000（背景）
- **纯白**: #FFFFFF（主文字、按钮）
- **深海蓝**: #0A0E27（卡片）
- **品牌蓝**: #4A90E2（强调）
- **浅灰**: #B8B8B8（次要文字）
- **错误红**: #FF4444

### 5.2 字体系统
- **字号**: 12-56px
- **字重**: 100-400（超轻到常规）
- **字间距**: 0.5-16px

### 5.3 组件规范

**按钮**
- 圆角: 12px
- 高度: 48px
- 主按钮: 白底黑字
- 次要按钮: 半透明白底白字

**输入框**
- 圆角: 12px
- 高度: 48px
- 背景: 半透明白(12%)

**卡片**
- 圆角: 24px
- 背景: 深海蓝
- 无边框

**弹窗**
- 底部弹出
- 圆角: 24px（顶部）
- 背景: 深海蓝

**Toast**
- 底部浮动（距底80px）
- 圆角: 12px
- 持续: 2秒

### 5.4 动画规范
- 页面切换: 300ms
- 按钮点击: 200ms
- 弹窗动画: 250ms
- 光球呼吸: 3000ms循环
