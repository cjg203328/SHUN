## 🚀 新的智能打包流程

### 使用新的 build.bat 脚本

1. **修改版本号**（每次更新前）
   ```batch
   REM 打开 build.bat，修改第8行
   set CURRENT_VERSION=1.0.3
   ```

2. **运行打包脚本**
   ```batch
   build.bat
   ```

3. **自动完成的操作**
   - ✅ 清理构建缓存
   - ✅ 获取依赖包
   - ✅ 构建APK
   - ✅ 复制APK到 `apk-output/瞬-v1.0.3-release.apk`
   - ✅ 自动创建 `versions/v1.0.3/` 目录
   - ✅ 自动生成版本文档模板：
     - README.md
     - 瞬-v1.0.3-版本说明.md
     - 瞬-v1.0.3-Bug修复记录.md
     - 瞬-v1.0.3-测试报告.md

4. **手动完成的操作**
   - 📝 填写版本文档内容
   - 📝 更新 `versions/CHANGELOG.md`
   - 📝 更新 `versions/版本索引.md`
   - 📱 测试APK功能

## 🔄 版本迭代流程

### 每次更新源码后

1. **修改源码** → 在 `flutter-app/lib/` 中修改代码

2. **更新版本号** → 修改 `build.bat` 中的 `CURRENT_VERSION`

3. **运行打包** → 双击 `build.bat`

4. **自动生成**
   - APK文件：`apk-output/瞬-v1.0.X-release.apk`
   - 版本目录：`versions/v1.0.X/`
   - 文档模板：自动创建4个文档模板

5. **手动补充**
   - 填写版本说明（新功能、Bug修复、优化改进）
   - 填写Bug修复记录
   - 填写测试报告
   - 更新CHANGELOG.md
