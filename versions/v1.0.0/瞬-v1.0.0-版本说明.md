## 📝 使用说明

### 登录
1. 输入11位手机号
2. 点击"发送验证码"
3. 输入验证码：123456
4. 点击"登录"

### 匹配
1. 进入"匹配"页面
2. 点击"开始匹配"
3. 等待2.5秒
4. 选择招呼语或自定义输入
5. 点击"发送"或"换一个"

### 聊天
1. 发送招呼后进入聊天
2. 输入消息（最多300字）
3. 点击发送按钮
4. 等待对方回复（Mock自动回复2秒后）
5. 查看倒计时，珍惜对话时间

### 消息列表
1. 查看所有进行中的对话
2. 点击进入聊天
3. 左滑删除对话

### 个人中心
1. 查看用户信息
2. 点击"关于瞬聊"查看介绍
3. 点击"退出登录"退出账号
