## 一、产品概述

### 1.1 产品定位
瞬是一款极简、克制的24小时限时匿名社交应用。

### 1.2 核心理念
- **限时性**: 24小时后对话自动消失
- **匿名性**: 渐进式身份解锁
- **克制性**: 每日20次匹配机会
- **纯粹性**: 极简设计，专注对话

### 1.3 目标用户
- 年龄: 18-35岁
- 特征: 喜欢深度交流，追求真实连接
- 场景: 深夜失眠、情绪低落、想找人聊天

## 三、功能详细设计

### 3.1 好友系统流程

```
陌生人匹配 → 开始聊天
  ↓
聊天15分钟 → 解锁头像
  ↓
聊天30分钟 → 解锁主页 + 可以添加好友
  ↓
点击"更多"→"添加好友"
  ↓
填写附言（可选）→ 发送请求
  ↓
对方收到通知（红点）
  ↓
对方接受 → 成为好友
  ↓
好友聊天（无时间限制）
```

### 3.2 权限管理流程

```
触发功能 → 检测权限状态
  ↓
未授权 → 显示权限说明弹窗
  ↓
用户点击"去开启" → 系统权限弹窗
  ↓
授权成功 → 执行功能
  ↓
永久拒绝 → 显示"去设置"弹窗
```

### 3.3 匹配消耗逻辑

```
点击"开始匹配" → 匹配中（2.5秒）
  ↓
  可以点"取消" → 不消耗次数 ✅
  ↓
匹配成功（弹出卡片） → 立即消耗1次 ✅
  ↓
  点"算了" → 不再消耗（已消耗）✅
  点"发送" → 不再消耗（已消耗）✅
```

## 四、设计规范

### 4.1 颜色系统
- **纯黑**: #000000（背景）
- **纯白**: #FFFFFF（主文字、按钮）
- **深海蓝**: #0A0E27（卡片）
- **品牌蓝**: #4A90E2（强调）
- **浅灰**: #B8B8B8（次要文字）

### 4.2 字体系统
- **字重**: 100-400（超轻到常规）
- **字间距**: 0.5-16px
- **字号**: 12-56px

### 4.3 组件规范
- **按钮**: 圆角12px，半透明背景
- **弹窗**: 底部弹出，圆角24px
- **提示**: 底部浮动，圆角12px
- **卡片**: 半透明背景，无边框
