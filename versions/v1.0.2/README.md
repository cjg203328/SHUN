## 主要更新

### ✨ 新增功能
1. **亲密度系统**
   - 替代时间进度条，通过消息互动提升亲密度
   - 5个亲密度等级，层层解锁用户信息
   - 可视化UI展示亲密度变化
   - 35分钟（250分）后可互关

2. **取关功能**
   - 支持单方面取关
   - 取关后对方只能发3条消息
   - 需要确认后才能恢复聊天

3. **消息状态管理**
   - 发送中状态（加载动画）
   - 发送失败状态（感叹号）
   - 支持重发功能
   - 完全参考微信设计

4. **位置权限优化**
   - 首次登录时获取，然后缓存记忆
   - 退出登录时清除缓存
   - 位置不是强依赖，没有位置也能匹配
   - 有位置权限优先匹配附近的人

5. **用户协议和隐私政策**
   - 登录页可点击查看完整协议
   - 符合应用商店上线标准

### 🐛 Bug修复
1. 修复打招呼后无法跳转到聊天页面
2. 修复设置页面无法返回
3. 修复所有子页面的导航返回逻辑
4. 修复聊天页面返回时回到错误的Tab

### 🔧 优化改进
1. 统一使用context.push()进行页面跳转
2. 优化导航栈管理
3. 完善权限管理逻辑
4. 提升用户体验流畅度
