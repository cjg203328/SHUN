# 瞬 v1.0.3

## 版本定位

- 版本号：`v1.0.3`
- 当前代码版本：`1.0.3+4`
- 首次发布日期：`2026-02-24`
- 当前状态：在 `v1.0.3` 基础上继续做稳定性和联调优化

## 本版本核心变化

- 个人资料刷新和本地持久化能力完善
- 权限管理与位置权限缓存逻辑调整
- 主导航、聊天返回链路修正
- 消息模型扩展到图片等类型
- 真实后端联调能力补齐：
  - OTP 登录
  - 设置同步
  - 好友关系
  - 匹配配额
  - 会话 REST
  - 聊天 WebSocket
  - 图片上传与媒体访问映射
  - 通知中心和未读角标
