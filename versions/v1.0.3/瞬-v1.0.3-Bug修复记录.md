## 🐛 Bug列表

### Bug #1: Message对象创建不完整
- **发现时间**：2026-02-24 19:00
- **严重程度**：中
- **影响范围**：消息发送功能
- **问题描述**：
  - 在ChatProvider中创建Message对象时，没有使用新增的type字段
  - 导致消息类型判断可能出错
  - 影响后续图片消息功能的实现
  
- **复现步骤**：
  1. 查看ChatProvider的sendMessage方法
  2. 发现创建Message时只传入了基础字段
  3. 缺少type、imagePath等新字段
  
- **修复方案**：
  - 在sendMessage方法中添加 `type: MessageType.text`
  - 在_mockReply方法中添加 `type: MessageType.text`
  - 使用copyWith方法替代手动创建Message对象
  
- **修复文件**：
  - `lib/providers/chat_provider.dart`
  
- **测试结果**：✅ 已修复，消息创建正常

### Bug #3: 导航返回逻辑错误
- **发现时间**：2026-02-24（之前版本）
- **严重程度**：高
- **影响范围**：所有聊天页面返回
- **问题描述**：
  - 从消息页进入聊天，返回时回到匹配页
  - 从好友页进入聊天，返回时回到匹配页
  - 用户体验不佳
  
- **复现步骤**：
  1. 在消息页点击某个对话
  2. 进入聊天页面
  3. 点击返回
  4. 发现回到了匹配页而不是消息页
  
- **修复方案**：
  - 在跳转时使用.then()回调
  - 返回后使用context.go()切换到正确的Tab
  - 匹配页 → tab=0
  - 消息页 → tab=1
  - 好友页 → tab=2
  
- **修复文件**：
  - `lib/widgets/match_tab.dart`
  - `lib/widgets/messages_tab.dart`
  - `lib/widgets/friends_tab.dart`
  - `lib/config/routes.dart`
  - `lib/screens/main_screen.dart`
  
- **测试结果**：✅ 已修复，返回逻辑正常
