# 瞬 v1.0.3 Bug修复记录

## 已完成的关键修复

### 第一批基础修复

- 修复 `Message` 创建字段不完整的问题
- 统一消息状态更新为 `copyWith` 风格，减少遗漏字段风险
- 修复聊天页返回到错误 Tab 的问题
- 修复个人资料修改后 UI 不刷新的问题

### 第二批交互与链路修复

- 优化位置权限缓存为会话级逻辑
- 修复部分弹窗层级与页面遮挡问题
- 优化匹配成功页和聊天输入区的交互层次
- 收敛导航和页面切换路径

### 第三批联调与发布路径修复

- 限制发布路径下的本地伪登录 fallback
- 限制发布路径下的本地伪匹配 fallback
- 限制发布路径下的离线消息伪发送成功
- 补齐媒体上传和媒体访问地址映射

## 当前仍需继续处理

- 测试环境中的权限插件 mock
- Smoke test 中的 Provider 和 GoRouter 包装缺失
- 部分测试仍引用已删除的旧设置接口
- `ChatProvider` 文件复杂度仍偏高

## 结论

`v1.0.3` 的修复重点已经从单点 Bug，转向“链路真实性、状态一致性、发布路径安全性”。后续版本应继续围绕这些点推进。
