# 瞬 v1.0.3 版本说明

## 版本信息

- 版本号：`v1.0.3`
- 当前代码版本：`1.0.3+4`
- 首次发布日期：`2026-02-24`
- 版本类型：稳定化迭代版本

## 版本目标

在 `v1.0.2` 基础上，把项目从“本地功能版”推进到“可持续联调版”，重点补齐资料、设置、匹配、聊天和媒体链路。

## 主要变更

### 客户端能力

- 个人资料修改后即时刷新
- 聊天返回路径和主导航行为修正
- 权限管理逻辑完善
- 夜间主题固定，日间分支收敛
- 通知中心和未读角标接入

### 聊天与媒体

- 消息模型支持文本和图片等扩展类型
- 会话、已读、WebSocket 实时收发接入
- 聊天图片、头像、背景图上传接入
- 媒体访问地址映射补齐

### 环境与发布

- 引入 `demo / development / staging / production` 环境识别
- 发布路径开始限制本地 fallback 伪成功
- 增加错误采集和埋点入口

## 相对 v1.0.2 的变化

- 从偏本地逻辑，转向前后端联调逻辑
- 从单纯 UI/交互修复，转向稳定性和可发布能力补齐
- 从散落文档，转向版本目录统一归档

## 当前限制

- 正式推送 SDK 仍未接入
- 正式监控、日志上报未闭环
- 后端持久化仍在推进，尚未完全替代内存实现
- `flutter test` 当前仍有待修复项，详见测试报告
