# 瞬 v1.0.4 版本说明

## 版本信息

- 版本号：`v1.0.4`
- 当前代码版本：`1.0.4+5`
- 开发起始日期：`2026-03-12`
- 版本类型：工程稳定性迭代

## 本轮目标

把项目从“有测试但不稳定”推进到“能持续回归”的状态，为后续功能开发提供可用基线。

## 当前已纳入范围

- 权限插件测试环境兼容
- smoke test 路由上下文补齐
- 设置测试清理过期断言
- 透明主页和竖版背景状态规则收敛
- `ChatProvider` 按职责拆分，降低单文件复杂度
- 消息撤回规则下沉到 Provider，避免 UI 与业务规则分离
- 聊天实时链路补齐幂等与精确已读回执处理
- 聊天历史回补合并本地恢复信息，避免图片消息重进会话后丢失预览
- 聊天离线补偿增加近期已发送消息保护窗口，减少最终一致性延迟带来的消息闪失
- 聊天失败重试规则收口为“仅文本可重试”，避免图片消息被错误降级为文本发送
- 聊天历史回补支持把本地失败/发送中消息与远端成功版本归并，减少重复消息与状态错乱
- 聊天实时回推支持识别“自己发出的消息回环”，避免把自回环消息当成新的来信处理
- 聊天自回环成功事件已补齐亲密度积分逻辑，和 `ACK` 成功链路保持同一业务结果
- 聊天图片回补归并规则已收紧，避免连续图片发送时把远端成功消息误绑到错误本地图片
- 聊天线程已读现在会联动通知中心同线程消息通知，避免消息页和通知页状态分叉
- 聊天通知中心已改为按消息维度幂等写入，避免重复消息生成重复通知

## 预期结果

- `flutter test` 恢复到可持续维护状态
- 测试失败从“环境问题”收敛为“真实逻辑问题”
- 聊天链路的消息顺序、未读与已读状态更接近真实线上规则
- 后续功能迭代可以围绕 `v1.0.4` 持续推进
